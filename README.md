# Data Analytics & Business Intelligence Labs

This repository contains weekly lab materials, datasets, and dashboards for the Data Analytics and Business Intelligence module.

## 📅 Weekly Structure

- Week 1: Introduction to Data Analytics + Pandas Basics
- Week 2: Data Cleaning & Preprocessing
- Week 3: Data Visualisation
- Week 4: Advanced Analytics Concepts
- (continue...)

## 📂 What Each Week Contains
- 📊 Dataset(s)
- 📓 Google Colab Notebook
- 📈 Dashboard (Excel / Power BI)
- 📘 Instructions & Tasks

## ▶️ How to Use

1. Navigate to the required week folder
2. Open the notebook in Google Colab
3. Upload dataset when prompted
4. Follow step-by-step tasks

## 🧠 Skills Developed

- Data analysis using Python (Pandas)
- Data visualisation (Matplotlib)
- Business Intelligence concepts
- Data-driven decision making

## 🚀 Tools Used

- Python
- Google Colab
- Excel Dashboards
- (Optional) Power BI

## 📌 Note

This repository is designed for Level 4 students with beginner-level programming experience.
